# styling

- avoid excessive comments - be a minimalist
- avoid excessive whitespace - especcially at the end of lines

# rust

- use `cargo fmt` for formatting
- use `cargo clippy` for linting - especially for the stuff just edited
- IMPORTANT use idiomatic rust - suggest fixes for non-idiomatic code

# nix env

- use `nix develop` for projects with a `flake.nix`

# git commits

Absolutely avoid adding coauthored by lines - use single author commits only
- do not mention the fact that the changes were made with the help of an AI assistant in the commit message
- make the commit messages concise and human like
